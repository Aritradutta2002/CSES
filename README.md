# CSES Problem Set Solutions

Java solutions for [CSES Problem Set](https://cses.fi/problemset/)

## Structure

```
src/
├── introductory/          # Introductory Problems
├── sorting/               # Sorting and Searching
├── dp/                    # Dynamic Programming
├── graph/                 # Graph Algorithms
├── range/                 # Range Queries
├── tree/                  # Tree Algorithms
├── math/                  # Mathematics
├── string/                # String Algorithms
├── geometry/              # Geometry
├── advanced/              # Advanced Techniques
├── sliding/               # Sliding Window Problems
├── interactive/           # Interactive Problems
├── bitwise/               # Bitwise Operations
├── construction/          # Construction Problems
├── advgraph/              # Advanced Graph Problems
├── counting/              # Counting Problems
├── additional1/           # Additional Problems I
└── additional2/           # Additional Problems II
```

## Usage

Each problem has its own Java file. To run a solution:

```bash
cd src/<category>
javac ProblemName.java
java ProblemName < input.txt
```

Or use VS Code's Run button with the Java extension.

## Template

Each solution follows this template structure:
- Fast I/O using BufferedReader
- Main solution logic
- Helper methods as needed
